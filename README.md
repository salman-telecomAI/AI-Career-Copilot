# AI Career Copilot

An AI-powered Career Copilot designed for Telecom, Cloud and AI professionals.

## Features

- AI Job Matching
- Retrieval-Augmented Generation (RAG)
- Profile Knowledge Base
- AI CV Generator
- AI Cover Letter Generator
- WhatsApp Approval Workflow
- Application Tracking
- FastAPI Dashboard
- Docker Support

## Technology Stack

- Python 3.14
- FastAPI
- SQLite
- Gemini AI
- ChromaDB
- LangChain
- Docker
- GitHub
